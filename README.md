# Home Task From Elfsight 

### Local Deploy
- Clone the repository:    
`git clone https://github.com/Ablyakimov/RickAndMorty.git`
- Go to the cloned folder:    
`cd RickAndMorty`
- Install dependencies:      
`yarn`
or
`npm i`
- Start a project:      
`yarn start`
or
`npm run start`
